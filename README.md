# GalaxyViewer
A small galaxy viewer for the [TIC80](https://tic.computer/) fantasy console.

![Alt Text](https://github.com/kirby81/GalaxyViewer/blob/master/galaxy.gif)

### Controls
- &leftarrow;  &rightarrow; switch planet
- w toggle the transparency
